Railken\Amethyst\Serializers\PostSerializer

The serializer is used to serialize an entity, you can retrieve it from the data.

use Railken\Amethyst\Managers\PostManager;

$manager = new PostManager();

$serializer = $manager->getSerializer();

And use it so serialize an entity Retrieving an entity

$entity = $repository->findOneById(1);
$serializer->serialize($entity)->toArray(); // Returns an array

Extend the class

Create the new serializer in app/Serializers/PostSerializer

namespace App\Serializers;

use Railken\Amethyst\Serializers\PostSerializer as Serializer;

class PostSerializer extends Serializer {
    // ...
}

Update the file configs/amethyst.post with the new class

return [
    'data' => [
        'post' => [
            'serializer' => App\Serializers\PostSerializer::class,
        ],
    ]
];
